# BannerKings Redux - Traducción al Español

Traducción completa del mod **BannerKings Redux** para Mount & Blade II: Bannerlord al español.

## 📌 Información del Proyecto

- **Mod Original**: [BannerKings Redux](https://www.nexusmods.com/mountandblade2bannerlord/mods/4284)
- **Objetivo**: Proporcionar una traducción completa y de calidad en español
- **Estado**: En progreso

## 📁 Estructura del Proyecto

```
/
├── translations/           # Archivos de traducción
│   ├── xml/               # Archivos XML traducidos
│   ├── strings/           # Strings del juego
│   └── localization/      # Archivos de localización
├── docs/                  # Documentación
├── scripts/               # Scripts de utilidad
└── README.md             # Este archivo
```
